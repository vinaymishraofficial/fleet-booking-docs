Installation Guide

Deploy Fleet Booking System on your ERPNext instance in minutes with this comprehensive installation guide.

Prerequisites

Before You Begin

Ensure you have the following requirements met before installing Fleet Booking System.

System Requirements

  • ERPNext: Version 13.0 or higher
  • Frappe Framework: Version 13.0 or higher
  • Python: Version 3.7 or higher
  • Node.js: Version 14.0 or higher

Required Permissions

  • System Manager role in ERPNext
  • SSH access to your server
  • Git access to clone repositories

Optional Services

  • Google Maps API: For route planning and location services
  • SMTP Server: For email notifications

Installation Steps

1

Download Fleet Booking System

Clone the Fleet Booking System app to your bench:

bench get-app https://github.com/alfastack-in/fleet_booking.git

Or if you have access to a local repository:

bench get-app fleet_booking
2

Install on Your Site

Install the app on your ERPNext site:

bench --site [your-site-name] install-app fleet_booking

Replace [your-site-name] with your actual site name.

3

Restart Services

Restart bench to apply changes:

bench restart
4

Verify Installation

Log into your ERPNext instance and verify the installation:

  1. Go to Setup > Applications
  2. Look for "Fleet Booking System" in the installed apps list
  3. Check that the app status shows as "Installed"

Initial Configuration

Set Up Google Maps Integration

  1. Go to Fleet Booking System > Fleet Booking Settings
  2. Enter your Google Maps API key
  3. Enable the features you want to use
  4. Save the settings

Google Maps API Key

You'll need a Google Maps API key with the following APIs enabled:

  • Maps JavaScript API
  • Geocoding API
  • Directions API
  • Places API

Configure User Roles

Set up the following roles for your users:

Employee

Can create and submit fleet booking requests

Fleet Manager

Can approve requests and assign vehicles/drivers

Driver

Can access driver dashboard and manage trips

Dispatcher

Can monitor all fleet operations

Troubleshooting

Common Issues

Installation fails with permission error

Ensure you have proper permissions and try running with sudo if necessary:

sudo bench get-app fleet_booking

App not showing in ERPNext

Clear cache and reload:

bench --site [your-site-name] clear-cache
bench restart

Google Maps not loading

Check that your API key is valid and has the required APIs enabled. Verify the key in Fleet Booking Settings.
