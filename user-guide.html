Employee Workflow

Creating a Fleet Booking Request

  1. Navigate to Fleet Booking: Go to Fleet Booking System > Employee Fleet Booking Request
  2. Create New Request: Click "Add Employee Fleet Booking Request"
  3. Fill Basic Information:
    • Trip purpose and description
    • Preferred trip date and time
    • Number of passengers
    • Special requirements
  4. Set Route with Google Maps:
    • Use the interactive map to set pickup location
    • Add multiple stopovers if needed
    • Set final destination
    • Review the calculated route and distance
  5. Submit Request: Review all details and submit for approval

üí° Pro Tip

Use the route optimization feature to automatically arrange stopovers in the most efficient order, saving time and fuel costs.

Fleet Manager Workflow

Processing Booking Requests

  1. Review Pending Requests: Check Employee Fleet Booking Request list for "Pending" status
  2. Evaluate Request: Review trip details, route, and business justification
  3. Make Decision:
    • Approve: If request is valid and resources are available
    • Reject: If request doesn't meet criteria or resources unavailable
  4. Assign Resources (for approved requests):
    • Select appropriate vehicle from available fleet
    • Assign qualified driver
    • Set any special instructions

Vehicle and Driver Assignment

When assigning resources, consider:

  • Vehicle Type: Match vehicle size and features to trip requirements
  • Driver Availability: Ensure driver is available for the scheduled time
  • Driver Qualifications: Match driver skills to trip complexity
  • Fuel Efficiency: Choose appropriate vehicle for distance and route

Using the Driver Dashboard

  1. Access Dashboard: Go to Fleet Booking System > Driver Dashboard page
  2. Select Trip: Choose from assigned trips in the dropdown
  3. Review Route: Check pickup, stopovers, and destination details
  4. Start Trip: Click "Start Trip" to begin GPS tracking
  5. Monitor Progress: Watch real-time stats and location updates
  6. End Trip: Click "End Trip" when journey is complete

Best Practices

For Employees

  • Submit requests with adequate lead time
  • Provide clear trip purpose and justification
  • Use accurate pickup and drop locations
  • Include all necessary stopovers in the initial request
  • Update requests promptly if plans change

For Fleet Managers

  • Review requests promptly to enable planning
  • Consider fuel efficiency in vehicle assignments
  • Match driver skills to trip requirements
  • Monitor fleet utilization patterns
  • Maintain clear approval criteria

For Drivers

  • Check assigned trips before starting shifts
  • Use the driver dashboard for all trip operations
  • Keep GPS location services enabled
  • Update trip status promptly
  • Report any issues immediately

Common Issues & Solutions

Google Maps not loading

Ensure Google Maps API key is configured in Fleet Booking Settings and has required permissions.

Driver dashboard not tracking location

Check browser location permissions and ensure GPS is enabled on the device.

Cannot assign vehicle to request

Verify the request is in "Approved" status and check vehicle availability for the requested dates.

Email notifications not sent

Check SMTP configuration in ERPNext settings and verify notification templates are active.
